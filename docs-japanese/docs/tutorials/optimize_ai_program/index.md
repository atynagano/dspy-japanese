# DSPyによるAIプログラムの最適化

本節では、DPSyの強力な最適化機能に焦点を当て、各種最適化手法を用いてAIプログラムを体系的に改善する方法について解説します。これらのチュートリアルではプログラミング概念の説明は最小限に抑え、DPSyの最適化機能がいかにしてアプリケーションの品質と性能を自動的に向上させられるかを具体的に示します。

## 数学的推論タスク

### [数学的推論](../math/index.ipynb)
DPSyプログラムを数学的推論タスク向けに最適化する方法を学びます。このチュートリアルでは、最適化機能がより良いプロンプティング戦略や少数ショット例を発見することで、複雑な数学問題に対する処理性能を大幅に向上させる手法を紹介します。

## モデル最適化

### [分類タスクのファインチューニング](../classification_finetuning/index.ipynb)
DPSyのファインチューニング最適化機能を活用し、大規模言語モデルから得た知識をよりコンパクトで効率的なモデルに転移させる方法を習得します。プロンプト最適化からモデルのファインチューニングまで、分類タスクにおける完全なワークフローを解説します。

## 高度なツール統合

### [高度なツール活用](../tool_use/index.ipynb)
外部ツールやAPIを利用するAIプログラムの最適化手法を探ります。このチュートリアルでは、DPSyの最適化機能がツールの使用効率をどのように向上させられるかを示し、ツール呼び出しシナリオにおける精度と効率の両面での改善方法を解説します。

### [ファインチューニングによるエージェント最適化](../games/index.ipynb)
ファインチューニングを通じて複雑なエージェントベースシステムを最適化する方法を学びます。このチュートリアルでは、ゲームのようなインタラクティブ環境において、戦略的思考と適応能力が求められるエージェントの性能向上手法を具体的に示します。
