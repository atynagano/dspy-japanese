# 実世界における応用事例

本節では、DSPyが様々な分野やユースケースにおいてどのように活用可能であるかを具体的に示す。各チュートリアルでは、DSPyのモジュール型プログラミング手法を用いて、本番環境向けのAIシステムを構築する方法を解説する。

## 注目すべき応用事例

### 📄 [llms.txt形式のドキュメント自動生成](../llms_txt_generation/index.md)
コードベースを分析し、llms.txt規格に準拠した構造化LLMフレンドリーなドキュメントを生成するAI駆動型ドキュメント生成システムの構築方法を学ぶ。

**主要概念:** リポジトリ解析、メタプログラミング、ドキュメント生成

### 📧 [電子メール情報抽出システム](../email_extraction/index.md)
DSPyの構造化予測機能を活用し、メッセージの分類、エンティティ抽出、アクション項目の特定を行うインテリジェントな電子メール処理システムを構築する。

**主要概念:** 情報抽出、分類、テキスト処理

### 🧠 [Mem0連携による記憶保持型ReActエージェント](../mem0_react_agent/index.md)
DSPyのReActフレームワークとMem0統合技術を用いて、セッション間で文脈を保持したインタラクションが可能な会話型エージェントを開発する。

**主要概念:** 記憶システム、会話型AI、エージェントの持続性

### 💰 [Yahoo Financeを活用した金融分析](../yahoo_finance_react/index.md)
LangChainツール統合機能を利用し、リアルタイム市場データの取得、ニュースセンチメント分析、投資インサイト提供が可能な金融分析エージェントを開発する。

**主要概念:** ツール統合、金融データ、リアルタイム分析

### 💻 [ドキュメントからの自動コード生成](../sample_code_generation/index.md)
DSPyのインテリジェント解析機能を活用し、URLからドキュメントを自動取得し、任意のライブラリに対応した動作可能なコード例を生成するシステムを構築する。

**主要概念:** Webスクレイピング、ドキュメント解析、自動学習、コード生成

### 🎮 [クリエイティブなテキストベースAIゲームの開発](../ai_text_game/index.md)
DSPyのモジュール型プログラミング手法を用いて、動的なストーリーテリング、AI駆動型NPC、適応型ゲームプレイを備えたインタラクティブなテキストベースアドベンチャーゲームを作成する。

**主要概念：**インタラクティブ・ストーリーテリング、ゲーム状態管理、キャラクター成長システム、AI駆動型ナラティブ
