# DSPyを用いたAIプログラムの構築

本セクションでは、本番環境で使用可能なAIアプリケーションをDSPyで構築するための実践的なチュートリアルを提供しています。各チュートリアルでは実際の使用事例を紹介しながら、DSPyのモジュール型プログラミング手法を活用して、堅牢でメンテナンス性の高いAIシステムを構築する方法を詳しく解説しています。

## 主要アプリケーション

### [会話履歴の管理](../conversation_history/index.md)
DSPyアプリケーションにおける会話履歴の管理方法について学習します。

### [DSPyによるAIエージェントの構築](../customer_service_agent/index.ipynb)
複雑なカスタマーサービスシナリオに対応できるインテリジェントエージェントの構築方法を習得します。このチュートリアルでは、文脈理解や会話状態の維持、適切な応答生成が可能なエージェントの構築手順を解説します。

### [DSPyモジュールのカスタマイズによるAIアプリケーション構築](../custom_module/index.ipynb)
特定のニーズに合わせてカスタマイズ可能なDSPyモジュールの作成方法を学びます。異なるアプリケーション間で再利用・組み合わせ可能な、モジュール化されたコンポーネントを構築するパターンを習得できます。

## Retrieval-Augmented Generation（RAG）

### [Retrieval-Augmented Generation（RAG）](../rag/index.ipynb)
DSPyを用いたRAGシステムの基本原理を習得します。外部知識ソースを活用して質問応答を行うシステムを構築するため、検索機構と言語モデルを統合する方法を詳しく解説します。

### [エージェント型RAGシステムの構築](../agents/index.ipynb)
RAGシステムをさらに進化させ、`ReAct`エージェントベースのシステムを構築します。情報の取得タイミングと方法を合理的に判断できるシステムを構築することで、RAGシステムの知能性と適応性を向上させます。

### [マルチホップRAG](../multihop_search/index.ipynb)
複数の情報源を跨いで多段階の推論が可能な高度なRAGシステムを構築します。複雑な調査分析タスクに最適な、洗練されたRAGシステムの構築方法を学べます。

## 専門用途向けアプリケーション

### [エンティティ抽出](../entity_extraction/index.ipynb)
テキストから特定のエンティティを識別・抽出するシステムの構築方法を習得します。情報処理やデータ分析アプリケーションにおいて不可欠な技術です。

### [分類タスク](../classification/index.md)
DSPyのモジュール型アプローチを活用した堅牢なテキスト分類システムを構築します。トピック分類の具体例を通じて学習します。

### [プライバシー保護を考慮した委任処理](../papillon/index.md)
小規模なローカルモデルと高度な外部モデルを組み合わせることで、高い性能を維持しつつプライバシー制約を尊重するAIシステム構築の先進的手法を探求します。

## 高度な推論機能

### [思考プログラム生成](../program_of_thought/index.ipynb)
言語モデルの能力とプログラム的推論を統合し、複雑な問題を解決するためのコードを生成・実行可能なシステムの構築方法を学びます。

## マルチモーダルアプリケーション

### [画像生成プロンプトの反復改善](../image_generation_prompting/index.ipynb)
DSPyを活用して画像生成プロンプトを体系的に最適化し、より良いビジュアルコンテンツを作成する手法を習得します。

### [音声処理](../audio/index.ipynb)
DSPyを用いた音声処理アプリケーションの開発手法を学びます。音声コンテンツの理解・処理・生成が可能なシステム構築の基礎を習得できます。
