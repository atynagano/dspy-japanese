# DSPyの学習：概要解説

DSPyは非常にシンプルなAPIインターフェースを提供しており、短期間で習得可能です。しかしながら、新たなAIシステムを構築するプロセスはより自由度の高い反復的な開発作業であり、DSPyの各種ツールや設計パターンを組み合わせながら、自身の目的に最適なシステムを構築していく必要があります。DSPyにおけるAIシステム構築の主要な3段階は以下の通りです：

1) **DSPyプログラミング**：これはタスク定義、その制約条件の設定、いくつかのサンプルデータを用いた探索、そしてこれらの情報を基に初期パイプライン設計を行う段階です。

2) **DSPy評価**：システムが動作し始めた後のこの段階では、初期開発用データセットの収集、DSPy独自の評価指標の定義、そしてこれらを活用したより体系的なシステム改良を行います。

3) **DSPy最適化**：システムを評価する方法を確立した後、DSPyの最適化ツールを用いてプロンプトやプログラムのパラメータ（重みなど）を調整します。

DSPyの学習と適用は、この順序に従って行うことを推奨します。例えば、設計が不十分なプログラムや不適切な評価指標を用いて最適化処理を開始することは、非生産的なアプローチと言えます。
